# 数据准备依赖安装说明

## 已安装的依赖

✅ **yacs** - 配置管理库（已安装）

## 需要安装的依赖

运行 `make_nuscenes_labels.py` 脚本需要以下依赖：

### 必需依赖

1. **yacs** - 配置管理
   ```bash
   pip install yacs
   ```

2. **shapely** - 几何操作库
   ```bash
   pip install shapely
   ```

3. **python-dotenv** - 环境变量管理（可选，但推荐）
   ```bash
   pip install python-dotenv
   ```

4. **nuscenes-devkit** - nuScenes 开发工具包
   ```bash
   pip install nuscenes-devkit
   ```

### 已包含的依赖

以下依赖通常在 diffbev 环境中已安装：
- numpy
- PIL (Pillow)
- tqdm

## 快速安装

运行以下命令安装所有必需依赖：

```bash
cd /media/ldk950413/data0/DiffBEV
micromamba activate diffbev
pip install yacs shapely python-dotenv nuscenes-devkit
```

## 验证安装

```bash
python -c "import yacs; import shapely; import nuscenes; print('✓ 所有依赖已安装')"
```

## 使用数据准备脚本

安装完依赖后，运行：

```bash
bash prepare_nuscenes_bev_data.sh
```

脚本会自动：
1. 检查环境
2. 检查依赖
3. 设置环境变量
4. 运行数据预处理脚本
5. 生成 BEV 标注
