## 配置使用

### 使用预训练权重（推荐）

在配置文件中设置：

```python
pretrained="/media/ldk950413/data0/DiffBEV/pretrained/swin_tiny_patch4_window7_224.pth"
```

### 从头训练

如果不想使用预训练权重，可以设置为 `None`：

```python
pretrained=None
```

**注意**: 从头训练需要更多迭代次数才能收敛，建议使用预训练权重。
